📥 Panduan Penggunaan Detail (Unduhan Mudah dalam Tiga Langkah)

Langkah 1: Salin Tautan Video VK

  1. Buka VK.com
  2. Putar video publik yang ingin Anda unduh
  3. Salin URL lengkap dari bilah alamat browser Anda
    Contoh format:
    https://vk.com/video-123456_789012345
    atau:
    https://vk.com/video123456789_789012345

💡 Tip: Tautan biasanya berisi video + ID numerik. Pastikan video dapat diputar tanpa login, jika tidak tidak dapat diunduh.

Langkah 2: Tempel Tautan dan Analisis

  1. Kembali ke halaman pengunduh ini
  2. Tempel tautan ke kotak input di atas (fungsi analisis Anda akan diintegrasikan di sini di masa depan)
  3. Klik tombol 【Analisis Video】

Sistem akan otomatis:
• Ekstrak judul, durasi, thumbnail video
• Dapatkan semua kualitas yang tersedia (seperti 480P, 720P, 1080P)
• Hasilkan tautan unduhan langsung tanpa pengalihan

Langkah 3: Pilih Kualitas dan Unduh

  • Klik tombol kualitas yang diinginkan (misalnya "720P MP4")
  • Browser akan otomatis mulai mengunduh file video HD
  • File secara default disimpan di folder "Unduhan" perangkat

📌 Saran Penggunaan:

  • Menonton di ponsel? → Pilih 480P atau 720P untuk menghemat data
  • Mencadangkan materi? → Pilih kualitas asli 1080P
  • Belajar offline? → Setelah diunduh, tonton kembali kapan saja dengan pemutar lokal

⚙️ Penjelasan Teknis (Referensi untuk Pengembang)

  • Dibangun berdasarkan alat sumber terbuka yt-dlp (mendukung VK), kompatibel dengan protokol streaming video VK terbaru.
  • Memproses aliran DASH/HLS multi-kualitas VK secara otomatis, menggabungkannya menjadi satu file MP4.
  • Semua unduhan terhubung langsung ke CDN video VK, server ini tidak meneruskan atau menyimpan cache konten video apa pun.
  • Tautan langsung video biasanya berlaku 1-6 jam, harap unduh tepat waktu.
  • Tidak bergantung pada API VK, tidak memerlukan access_token, hanya menganalisis melalui metadata halaman web publik.

❓ Pertanyaan yang Sering Diajukan (FAQ)

T: Mengapa muncul pesan "Video tidak dapat dianalisis"?
Kemungkinan penyebab: Video dalam status pribadi, format tautan salah, video telah dihapus, atau VK sementara memperbarui mekanisme anti-scraping.
T: Tidak ada opsi 1080P?
VK hanya menyediakan 1080P untuk beberapa unggahan HD. Jika video asli maksimum 720P, opsi yang lebih tinggi tidak akan ditampilkan.
T: Dapatkah saya mengunduh video grup (Komunitas)?
Ya! Selama grup dan video dalam status publik, Anda dapat mengunduhnya secara normal.
T: Tidak dapat menemukan file setelah mengunduh di ponsel?
Android: Gunakan "Manajer File" untuk masuk ke folder "Unduhan"; iOS: Di Safari, klik unduhan lalu pilih "Simpan ke File" aplikasi.
T: Apakah mendukung siaran ulang live VK?
Beberapa siaran ulang live publik dapat diunduh, tetapi dibatasi oleh kebijakan retensi VK. Disarankan untuk mengunduh secepatnya.
