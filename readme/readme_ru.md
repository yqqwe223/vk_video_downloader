## 📋 Описание

**VK Video Info Tool** — это легковесное веб-приложение, разработанное для анализа и получения метаданных общедоступных видеозаписей на платформе VK (ВКонтакте). Инструмент помогает пользователям, исследователям и архивариусам получать техническую информацию о видеоконтенте, не вмешиваясь в инфраструктуру платформы и не обходя её механизмы защиты.

### ✨ Основные возможности

- 🔍 **Анализ ссылок**: Поддержка ввода публичных URL-адресов видеозаписей VK для запроса связанных метаданных
- 📊 **Отображение информации**: Показ названия, описания, продолжительности, доступного разрешения и даты публикации
- 🌐 **Многоязычный интерфейс**: Полная поддержка русского языка (для пользователей из России и СНГ)
- 📱 **Адаптивный дизайн**: Оптимизированный интерфейс для настольных компьютеров, планшетов и мобильных устройств
- ⚡ **Эффективная обработка**: Валидация на стороне клиента с оптимизированной коммуникацией через API
- 🔒 **Приоритет приватности**: Данные пользователей, история запросов и медиаконтент не сохраняются

## 📖 Инструкция по использованию

1. Скопируйте ссылку на видеозапись VK, которая имеет **публичный доступ**
2. Вставьте URL в поле ввода веб-интерфейса
3. Нажмите кнопку «Анализировать» для получения доступных метаданных
4. Используйте отображённую информацию в личных, исследовательских или архивных целях

> ⚠️ Инструмент работает исключительно с общедоступным контентом. Частные или ограниченные в доступе видеозаписи не могут быть обработаны по техническим и правовым причинам.

## ❓ Часто задаваемые вопросы

**В: Является ли использование этого инструмента бесплатным?**  
О: Да, это проект с открытым исходным кодом под лицензией MIT, который можно использовать свободно.

**В: Сохраняются ли видеофайлы временно на серверах?**  
О: Нет. На любом этапе процесса медиаконтент не кэшируется, не загружается и не хранится на наших системах.

**В: Работает ли инструмент с приватными видео VK?**  
О: Нет. По техническим причинам и в целях соблюдения законодательства поддерживаются только общедоступные материалы.

**В: Доступна ли техническая документация по API?**  
О: Внутренняя техническая спецификация может быть предоставлена по запросу для проверенных академических исследований.
