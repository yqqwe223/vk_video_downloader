## 📋 Descripción

**VK Video Info Tool** es una aplicación web ligera desarrollada para analizar y consultar metadatos de videos de acceso público en la plataforma VK (VKontakte). Esta herramienta permite a usuarios, investigadores y archivistas obtener información técnica sobre contenidos de video, sin interferir con la infraestructura ni los mecanismos de seguridad de la plataforma.

### ✨ Características Principales

- 🔍 **Análisis de Enlaces**: Soporte para ingresar URLs de videos públicos de VK y consultar sus metadatos asociados
- 📊 **Visualización de Información**: Muestra título, descripción, duración, resolución disponible y fecha de publicación
- 🌐 **Interfaz Multilingüe**: Soporte completo para el idioma español (España y Latinoamérica)
- 📱 **Diseño Responsivo**: Experiencia optimizada para escritorio, tablet y dispositivos móviles
- ⚡ **Procesamiento Eficiente**: Validación en el lado del cliente con comunicación API optimizada
- 🔒 **Privacidad Prioritaria**: No se almacenan datos de usuario, historiales de consulta ni contenidos multimedia

## 📖 Guía de Uso

1. Copie el enlace de un video de VK que sea **de acceso público**
2. Pegue la URL en el campo de entrada de la interfaz web
3. Haga clic en "Analizar" para recuperar los metadatos disponibles
4. Utilice la información mostrada con fines de referencia personal, investigación académica o preservación digital

> ⚠️ Esta herramienta funciona exclusivamente con contenidos de acceso público. Los videos privados o con restricciones de visibilidad no pueden ser procesados por limitaciones técnicas y de cumplimiento normativo.
