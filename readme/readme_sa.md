## 📋 الوصف

**أداة معلومات فيديوهات VK** هي تطبيق ويب خفيف الوزن تم تطويره لتحليل واستعلام البيانات الوصفية (Metadata) لمقاطع الفيديو المتاحة للعموم على منصة VK (فكونتاكتي). تُمكّن هذه الأداة المستخدمين والباحثين وأمناء الأرشيف من الحصول على معلومات تقنية حول محتوى الفيديو، دون التدخل في البنية التحتية للمنصة أو تجاوز آليات الحماية الخاصة بها.

### ✨ الميزات الرئيسية

- 🔍 **تحليل الروابط**: دعم إدخال عناوين URL العامة لفيديوهات VK للاستعلام عن بياناتها الوصفية المرتبطة
- 📊 **عرض المعلومات**: إظهار العنوان، الوصف، المدة الزمنية، دقة الفيديو المتاحة، وتاريخ النشر
- 🌐 **واجهة متعددة اللغات**: دعم كامل للغة العربية (للمستخدمين في المنطقة العربية والعالم)
- 📱 **تصميم متجاوب**: واجهة مستخدم محسّنة لأجهزة الكمبيوتر المكتبية والأجهزة اللوحية والهواتف الذكية
- ⚡ **معالجة فعّالة**: التحقق من صحة البيانات على جانب العميل مع تواصل محسّن عبر واجهة برمجة التطبيقات (API)
- 🔒 **الخصوصية أولاً**: لا يتم تخزين أي بيانات للمستخدمين أو سجلات الاستعلامات أو محتوى الوسائط

## 📖 دليل الاستخدام

1. انسخ رابط فيديو من منصة VK يكون **متاحاً للعموم**
2. ألصق عنوان URL في حقل الإدخال بواجهة الويب
3. انقر على زر "تحليل" لاسترداد البيانات الوصفية المتاحة
4. استخدم المعلومات المعروضة لأغراض المرجعية الشخصية، أو البحث الأكاديمي، أو الحفظ الرقمي

> ⚠️ تعمل هذه الأداة حصرياً مع المحتوى المتاح للعموم. لا يمكن معالجة الفيديوهات الخاصة أو المقيدة الوصول لأسباب تقنية والتزاماً باللوائح القانونية.

## ⚖️ الالتزام القانوني وتنويه هام

هذا المشروع:

- ✅ يحترم إرشادات `robots.txt` المحددة من قبل منصة VK
- ✅ يعالج فقط المعلومات المتاحة للعموم والمرئية دون الحاجة للمصادقة
- ✅ لا يخزن البيانات الشخصية، أو سجلات الاستعلامات، أو ملفات الوسائط
- ✅ موجّه حصرياً للأغراض التعليمية، والبحثية، وأغراض الحفظ الثقافي والأرشيفي
- ✅ لا ينفذ أي أساليب لتجاوز آليات الأمان، أو المصادقة، أو قيود الوصول

**هام**: يتحمل المستخدم المسؤولية الكاملة عن الاستخدام الذي يجرية لهذه الأداة. نوصي بالاطلاع المسبق على شروط استخدام منصة فكونتاكتي والتشريعات المعمول بها في مجال الملكية الفكرية ضمن نطاق اختصاصك القانوني.
