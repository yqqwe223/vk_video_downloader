## 📋 Descrição

**VK Video Info Tool** é uma aplicação web leve projetada para analisar e extrair metadados de vídeos publicamente disponíveis na plataforma VK (VKontakte). Esta ferramenta auxilia usuários e pesquisadores a obterem informações técnicas sobre conteúdos de vídeo, sem interferir na infraestrutura da plataforma.

### ✨ Principais Funcionalidades

- 🔍 **Análise de URL**: Suporte para inserção de links públicos de vídeos do VK para consulta de metadados
- 📊 **Visualização de Metadados**: Exibição de título, descrição, duração, resolução e data de publicação
- 🌐 **Interface Multilíngue**: Suporte completo ao idioma português (Brasil e Portugal)
- 📱 **Design Responsivo**: Interface otimizada para desktop, tablet e dispositivos móveis
- ⚡ **Processamento Eficiente**: Validação no lado do cliente com comunicação API otimizada
- 🔒 **Privacidade em Primeiro Lugar**: Nenhum dado do usuário ou histórico de consultas é armazenado

## 📖 Guia de Utilização

1. Copie o link de um vídeo do VK que seja **publicamente acessível**
2. Cole a URL no campo de entrada da interface web
3. Clique em "Analisar" para recuperar os metadados disponíveis
4. Utilize as informações exibidas para fins de referência pessoal ou pesquisa acadêmica

> ⚠️ Esta ferramenta funciona exclusivamente com vídeos de acesso público. Conteúdos privados ou restritos não podem ser processados por limitações técnicas e legais.
