## 📋 Beschreibung

**VK Video Info Tool** ist eine leichtgewichtige Web-Anwendung zur Analyse und Extraktion öffentlich zugänglicher Video-Metadaten von VK (VKontakte). Das Tool unterstützt Nutzer dabei, technische Informationen zu Videos abzurufen, ohne dabei in die Plattform-Infrastruktur einzugreifen.

### ✨ Hauptfunktionen

- 🔍 **URL-Analyse**: Eingabe von öffentlichen VK-Video-Links zur Metadaten-Abfrage
- 📊 **Metadaten-Anzeige**: Anzeige von Titel, Beschreibung, Dauer, Auflösung und Upload-Datum
- 🌐 **Mehrsprachige Oberfläche**: Vollständige Unterstützung der deutschen Sprache
- 📱 **Responsive Design**: Optimiert für Desktop, Tablet und Mobile
- ⚡ **Schnelle Verarbeitung**: Client-seitige Validierung mit effizienter API-Kommunikation
- 🔒 **Datenschutzfreundlich**: Keine Speicherung von Nutzerdaten oder Abfrageverläufen

## 📖 Nutzungshinweise

1. Kopieren Sie den Link eines **öffentlichen** VK-Videos
2. Fügen Sie die URL in das Eingabefeld der Web-Oberfläche ein
3. Klicken Sie auf "Analysieren", um verfügbare Metadaten abzurufen
4. Nutzen Sie die angezeigten Informationen für persönliche Referenzzwecke

> ⚠️ Das Tool funktioniert ausschließlich mit Videos, die öffentlich zugänglich sind. Private oder eingeschränkte Inhalte können nicht verarbeitet werden.
